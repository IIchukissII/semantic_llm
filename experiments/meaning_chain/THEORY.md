- `ν` = number of nouns processed
- `b/ν` = bond-to-noun ratio (saturation metric)
- `(b/ν)_max` = theoretical maximum ratio
- `τ_ν` = time constant in noun units
- `e` = Euler's number ≈ 2.71828

**LaTeX form:**
$$\frac{b}{\nu} = \left(\frac{b}{\nu}\right)_{\max} \cdot \left(1 - e^{-\nu/\tau_\nu}\right)$$

## 7. Boltzmann Transitions

> ✓ **STATUS: VALIDATED** (E4 test, kT = 0.82, error 1.2%)

### The Core Law

Transition probability between concepts follows Boltzmann statistics:

```
P(A → B) ∝ exp(-|Δτ| / kT)
```

Where:
- `Δτ = τ_B - τ_A` (abstraction difference)
- `kT = 0.82` (natural semantic temperature, measured)

### Why Boltzmann?

Just as particles prefer low-energy states, meanings prefer:
- Nearby abstraction levels (small |Δτ|)
- Smooth transitions over jumps

### Temperature Interpretation

```
kT = 0.82 (measured empirically)

Note: 1 - 1/e ≈ 0.632, so kT ≈ 0.82 is HIGHER than this.
      The relationship to e remains an open question.

Low T:  Stays at same abstraction level
        Conservative, focused navigation

High T: Jumps across abstraction levels
        Exploratory, creative navigation
```

### The Dynamic Effect

Static τ-levels (from Part I) become dynamic through Boltzmann:

```
STATIC: τ₁, τ₂, τ₃, ...  exist as positions
DYNAMIC: P(τ₁ → τ₂) = exp(-|Δτ|/kT)  governs movement
```

```

Remaining after k clusters: (1/e)^k
```

- `b` = unique bonds
- `ν` = nouns processed
- `τ_ν` = time constant (nouns until 63.2% saturation)

